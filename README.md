# Face Emotion Detection System

## 📌 Description
This project detects human emotions in real-time using a CNN model trained on the FER2013 dataset.

## 🧠 Technologies Used
- Python
- OpenCV
- TensorFlow / Keras

## 🚀 Features
- Real-time face detection
- Emotion classification
- Trained deep learning model

## ▶️ How to Run
1. Install requirements
2. Run:
   python detect_emotion.py

## 📊 Dataset
FER2013 (Kaggle)
